# 1. Project Objective

The objective of the **ProSpy** project is to develop a **Machine Learning-based system** capable of detecting **fake Instagram profiles** by analysing profile information, behavioural patterns, and account activities.

The system classifies Instagram accounts as **Genuine** or **Fake** using various profile-level and behavioural features. The primary goal is to improve social media security by identifying spam accounts, bot accounts, impersonation profiles, and malicious users.

# 3. Introduction

Instagram is one of the world's largest social networking platforms. Millions of users interact daily through photos, videos, reels, and messaging. Due to its popularity, Instagram has become a major target for fake accounts, spam bots, phishing campaigns, and impersonation attacks.

Fake Instagram accounts are commonly used for:

- Identity Theft
- Spam Marketing
- Fake Followers
- Online Scams
- Phishing
- Fake Giveaways
- Cryptocurrency Fraud
- Misinformation

The **ProSpy** project aims to detect these fake Instagram accounts using Machine Learning techniques by analysing profile attributes and behavioural characteristics.

# 5. Selected Instagram Profile Features

| Feature | Importance |
|----------|------------|
| has_profile_pic | Detects suspicious or missing profile pictures commonly used by fake Instagram accounts. |
| bio_length | Helps identify incomplete, empty, or spam-like biographies. |
| username_randomness | Random usernames usually indicate bot-generated accounts. |
| followers | Extremely low or abnormal follower counts indicate suspicious behaviour. |
| following | Fake accounts generally follow thousands of users. |
| follower_following_ratio | A poor follower-following ratio indicates possible fake activity. |
| account_age_days | Newly created Instagram accounts are more likely to be fake. |
| posts | Very few or extremely high posts indicate suspicious activity. |
| posts_per_day | Detects automated posting behaviour. |
| spam_comments_rate | Measures spam interaction generated by the account. |
| suspicious_links_in_bio | Detects phishing or malicious URLs. |
| verified | Verified Instagram accounts are generally more trustworthy. |

### Why These Features?

These behavioural and profile-based features provide valuable information for distinguishing genuine Instagram users from fake, spam, or automated bot accounts.

# 6. Fake Instagram Profile Indicators

| Indicator | Reason |
|-----------|--------|
| Profile picture appears stolen or AI-generated | Fake accounts frequently use stolen or AI-generated images. |
| Very few uploaded posts | Genuine users usually maintain a posting history. |
| Recently created account | Newly created accounts have a higher probability of being fake. |
| Very few Instagram followers | Fake accounts generally have very limited followers. |
| Unusual follower-following ratio | Following many users while having few followers is suspicious. |
| Incomplete profile information | Missing profile information often indicates fake accounts. |
| Generic or copied Instagram bio | Bots frequently reuse identical biographies. |
| Posts uploaded within a short period | Indicates automated profile creation. |
| Generic or repetitive comments | Suggests automated commenting behaviour. |
| High spam comment rate | Strong indication of spam activity. |
| Suspicious links in bio | May redirect users to phishing or scam websites. |
| No tagged Instagram photos | Genuine users usually appear in tagged photos. |
| Avoids identity verification | Fake users often refuse verification. |
| Requests money | Financial requests indicate possible scams. |
| Claims without evidence | Fake users often pretend to be celebrities or professionals. |
| Instagram activity does not match claimed location | Indicates inconsistent identity. |
| Inconsistent profile images | May indicate stolen photographs. |
| Writing style changes frequently | Could indicate multiple operators or AI-generated messages. |
| Avoids personal questions | Difficulties maintaining a fake identity. |
| Reverse image search matches another account | Strong evidence of impersonation. |
